# 调参速查 (一页搞定)

> 配 GUI 曲线 tab 看. 改 `config/control.yaml` 后, **保存即生效, 无需重编**.

## 看什么图

| 图 | 关键看点 |
|----|---------|
| **Yaw / Pitch** | `cmd` (绿) 和 `state` (蓝) 是否贴合; `error` (红) 是否快速归零 |
| **target_u / v** | 像素误差; 静止目标时应贴近 (0,0) |
| **bbox_h** | 目标大小 ≈ 距离反比; 小 = 远距离 |
| **FPS** | < 60 说明系统卡, 调参意义不大 |

## 三步法 (按顺序调, 不要跳)

```
1. 关 ki/ff, 调 kp 直到 error 收敛快不振荡
2. 加 ff (前馈) 让 cmd 提前 state 一点, 减小 error
3. 加 ki (积分) 消除稳态误差 (匀速跟随时 error 不归零)
```

## 症状 → 改哪个

### 跟得慢 / 追不上 / `state` 总落后 `cmd`
| 优先级 | 参数 | 调法 |
|-------|-----|------|
| 1 | `kp_yaw` / `kp_pitch` | +1.0 直到刚开始抖, 退回一档 |
| 2 | `ff_gain_yaw` / `ff_gain_pitch` | 0.0 → 1.0 (有 Kalman 速度时直接 1.0) |
| 3 | `lowpass_alpha_yaw` / `_pitch` | 提高 (0.92 → 0.95), 越接近 1 越快 |
| 4 | `pitch_rate_assist_kp` | pitch 追不上时 +0.5 |

### 振荡 / 抖 / 来回晃
| 优先级 | 参数 | 调法 |
|-------|-----|------|
| 1 | `kp_yaw` / `kp_pitch` | -1.0 |
| 2 | `damping_kd` | +0.005 (0.008 → 0.013) |
| 3 | `lowpass_alpha_*` | 降低 (0.92 → 0.85) |
| 4 | `cmd_max_yaw_step_deg` | 减小 (10 → 6) 限单帧跳变 |

### 匀速跟踪有固定误差 (`error` 不归零)
| 参数 | 调法 |
|-----|------|
| `ki_yaw` / `ki_pitch` | 0 → 0.3, 每次 +0.1 |
| `ki_max_deg` | 误差大时调大 (3 → 5), 但不要超过 10 |

> ⚠ ki 太大会过冲. 先确认 kp + ff 已经调好再开 ki.

### 远距离瞄不准 / 近距离过冲
开 `adaptive_kp: 1`, 然后:
- `kp_far` (远) > `kp_near` (近); 一般差 1.5–2 倍
- 看 bbox_h 实际范围调阈值: `kp_far_area` (远端 bbox 小) / `kp_near_area`

### 目标丢了云台乱飞
| 参数 | 含义 |
|-----|------|
| `lost_yaw_scan_amp` | 丢失后 yaw 扫多大 (度), 太大会看不见目标 |
| `lost_pitch_max` / `_min` | 抬/低头硬上限, 防止云台撞限位 |
| `scan_enter_delay_ms` | 丢多久才开始扫, 大 = 不那么慌 |

### 启动时云台疯抖一下
| 参数 | 调法 |
|-----|------|
| `startup_prep_ms` | 加大 (1000 → 2000) 给电机更长稳定时间 |
| `startup_home_tol_deg` | 调大 (1.0 → 2.0) 放宽对中精度 |

## 不要碰 (除非你知道在做什么)

- `R_camera2gimbal` / `t_camera2gimbal` — 相机云台外参, 标定来的
- `boresight_yaw_deg` / `boresight_pitch_deg` — 激光偏置, 用 `tools/laser_calib_gui.py` 自动学
- `target_height_m` / `target_width_m` — 物理尺寸, 改了会影响测距
- `yaw_sign` / `pitch_sign` — 方向反了会瞄反, 一开始定好就别动

## 一键测试流程

```bash
# 1. 启动 tracking (调参模式), 桌面摆个静止目标
./build/bin/tracking --no-gimbal     # 先关云台测视觉
./build/bin/tracking                 # 再开云台

# 2. GUI 看曲线, 故意让目标平移 / 远近变化, 观察 cmd vs state vs error
./build/bin/control_panel

# 3. 改 yaml, Ctrl+S, 不需要重启 tracking, 下一帧就生效
```
