## 1. 雷达系统概述

雷达可自主获取战场信息，并通过多机通信向己方机器人或裁判系统选手端发送信息，还可以**截获对方的信息或反制对方空中机器人的空中支援**。

| 关键特征 | 说明 |
|---------|------|
| 初始区域 | 雷达基座 |
| 运行方式 | 自动运行，最多配置 1 个遥控器用于调试 |
| 多机通信 | 可以向所有己方机器人发送数据，但只能接收哨兵机器人的数据 |

雷达的功能包括：**雷达识别**、**雷达反制**、**雷达解析信息波**。

---

## 2. 雷达系统组成

### 2.1 整体结构

雷达由**运算平台端**与**传感器端**构成，两者需通过电缆连接。雷达需具备运算平台端与传感器端，且能够显示调试画面。

> **雷达传感器端与雷达运算平台端安装位置距离较远，建议参赛队伍连接电缆预留至少 5m 的有效长度。**

- 雷达运算平台端的运算平台可以使用成品模组，不受第三方成品模组限制

## 5. 传感器端（小云台摆放位置）

### 5.1 传感器端制作参数（表 2-14）

| 项目 | 限制 | 备注 |
|------|------|------|
| 最大伸展尺寸（长×宽） | 不得超出雷达传感器端放置平台 | 用于连接的线材不计入尺寸 |
| 最大高度 | 任意情况下不得大于 **1500mm** | 建议传感器支架高度在 **1.2m 以上** |

| 裁判系统模块 | 无 | - |

> 雷达传感器端所涵盖的无线电接收装置、激光发射装置可以与雷达传感器端主体分离，但长、宽不得超出雷达传感器端的放置平台。

### 5.2 传感器端安装规范

- S131：传感器端需**固定在雷达传感器支架上并放置于雷达基座上**
- S132：**激光发射装置、无线接收装置需安装在雷达传感器端**
- S133：参赛队伍需**自行设计雷达传感器支架**，以提高传感器安装高度
- S134：雷达传感器支架的尺寸需满足安置于雷达基座安装平面上（参照比赛规则手册关于雷达基座描述）
- **传感器端的信号传输与电源需由参赛队伍自行解决**
- 参赛队伍需自行为设备安装保护罩，防止比赛中被弹丸击打造成损伤

### 5.3 小云台（激光发射装置执行机构）

雷达的激光发射装置需要具备**执行机构（云台）**来瞄准和跟踪空中机器人。执行机构的相关规则：

- 激光发射装置及执行机构安装在传感器端
- 电源管理模块的 **Gimbal 接口**为雷达的**激光发射装置及其执行机构**供电（而非底盘供电）
- 仅在对方空中机器人发起空中支援时，雷达系统上的激光发射装置及其执行机构**才会上电**
- 雷达被判罚下时，激光发射装置及其执行机构将被断电

## 6. 激光发射装置规格

### 6.1 基本规格（S25-S30）

| 参数 | 限制/建议 |
|------|-----------|
| 安装位置 | 仅可安装在**雷达传感器端** |
| 安装数量 | 仅允许安装 **1 个** |
| 实测功率 | 需小于 **1mW** |
| 波段 | 不得大于 **730nm**（建议范围 600-700nm） |
| 波形建议 | 建议发射**连续波（CW）** |
| 有效光斑直径 | 建议不小于 **5mm** |
| 光斑功率密度 | 建议不低于 **3.82 μW/mm²** |
| 激光安全等级 | 允许 Class I（不建议）、Class II；不允许 Class III 及以上 |
| 触发距离限制 | 不得在直线距离 1000mm 外触发其他机器人上的激光检测模块 |

> S25：激光发射装置特指雷达上安装的用于触发裁判系统激光检测模块的激光产品，仅可安装在雷达传感器端。其他机器人上安装的所有红外光源或激光产品不得在直线距离 1000mm 外触发激光检测模块。

### 6.2 注意事项

- S26：不得改装激光产品。安全标准认定取以下最高值：
  - 激光产品自带的安全铭牌数值
  - 厂家/经销商出具的相关证明
  - 检录人员使用激光功率计紧贴激光产品所测得的实测功率
- S28：使用任何光学手段都不得对任何人员造成身体伤害

## 7. 接线与供电

### 7.1 主控模块与电源管理模块连接

正式比赛时，雷达需参照组委会指定的方式连接电源管理模块与主控模块。

**连接流程：**

1. 战场提供有线方式使得雷达连入战场网络和服务器
2. 雷达运算平台端方式平台提供**有线接入点：单端母头航空线**
3. 场地人员需将雷达运算平台端上电源管理模块中的**主控接口**，通过官方的**航空对接线**连接到有线接入点上

**接线要求：**

- 在比赛过程中，雷达**无需自身携带主控**，检录前请取下机器人上调试用的主控
- 雷达的**电源管理模块必须外露**，且放在工作人员容易操作的位置
- 雷达电源管理模块上的**主控接口不可连接任何设备**
- 该航空线需理顺收纳，不可相互缠绕，且应易于拉出并连接在有线接入点上
- S141：在设计雷达结构时，需将**电源管理模块的主控模块接口外露**

## 8. 雷达反制空中机器人机制

### 8.1 反制原理

雷达系统上可安装激光发射装置，用于**瞄准和照射空中机器人上的激光检测模块**。

- **仅在对方空中机器人发起空中支援时**，己方雷达系统上的激光发射装置及其执行机构上电
- 空中机器人具有"被瞄准进度" **P**，取值范围 0~100

### 8.2 反制效果

| 条件 | P 值 | 效果 |
|------|------|------|
| P 达到 P0（首次为 50） | P = 50 | 空中机器人发射机构被**锁定**，P 归零，锁定状态持续 **45 秒**后解除 |
| 每局最多锁定次数 | - | **3 次** |

### 8.3 锁定难度递增机制

每次成功锁定后，对方空中机器人激光检测模块的被瞄准难度提高：

| 锁定次数 | 激光检测模块变化 | P0 值 |
|---------|-----------------|-------|
| 第 1 次锁定前 | 无特殊变化 | **50** |
| 第 2 次锁定前 | 可检测区域纵向缩减为原面积 **3/5** | **100** |
| 第 3 次锁定前 | 可检测区域纵向缩减为原面积 **1/5**，且**不再主动发光** | **100** |

### 8.4 "被瞄准进度"计算逻辑

**参数设定：**

- t：当前激光连续照射的时长
- Δt：判定周期，固定为 **0.1 秒**
- n：连续判定次数（照射开始时 n=0，每触发一次判定 n 加 1）

**判定规则：**

- 当空中机器人激光检测模块**未被激光照射**时，进度 P 以速率 **0.5/s** 匀速衰减（但不会小于 0），同时 t、n 立即归零
- 当空中机器人激光检测模块**被激光照射**时，P 停止衰减。每当 t 累计满 0.1 秒，触发一次进度增加：
  - 第 1 个 0.1 秒：P = P + 1
  - 第 2 个 0.1 秒：P = P + 2
  - 第 n 个 0.1 秒：P = P + n
- 若单次持续照射时间不足 0.1 秒即中断，则 t、n 立即归零

### 8.5 相关离线判罚

- **激光检测模块离线**（空中机器人）：在空中支援期间，始终视为被对方激光发射装置**持续照射**
- **雷达被判罚下**：激光发射装置及其执行机构将被**断电**，且多机通信被断开（仅能单向接收，不再能发送）

## 9. 雷达识别与双倍易伤机制

### 9.1 被标记进度 P 的计算

| 标记状态 | x 的更新逻辑 |
|---------|------------|
| 准确（d < 0.8m） | 若上一次为准确或半准确，x = x + 1.0；否则 x = 1.0 |
| 半准确（0.8m ≤ d < 1.6m） | 若上一次为准确或半准确，x = x + 0.5；否则 x = 0.5 |
| 错误（d ≥ 1.6m） | 若上一次错误/中断，x = x − 0.8；否则 x = −0.8 |
| 数据中断 | 每累计 0.5 秒未收到数据，视为触发一次"错误"判定 |

P 更新公式：**P = P + x**（0 ≤ P ≤ 150）

### 9.2 双倍易伤机制

- 当雷达每累计使对方机器人易伤 **1 分钟**（同时有多台机器人易伤时，时间不累加），获得 1 次触发机会
- 消耗机会后，所有来源于雷达的易伤数值**翻倍**，持续 **30 秒**
- 每局比赛中，雷达至多触发 **2 次**"双倍易伤"

| 评分项目 | 分值 |
